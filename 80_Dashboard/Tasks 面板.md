## ✅ 今日任务汇总
> 包含：今天新建、今天截止、今天开始、今天计划、今天完成

```tasks
( created on today ) OR ( due on today ) OR ( starts on today ) OR ( scheduled on today ) OR ( done on today )
hide task count
```

## ⚠️ 历史遗留 (过期未完成)
> 截止日期在 **Today** 之前，且尚未完成的任务

```tasks
due before today
hide task count
```

## 🔥 未来高优任务
> 截止日期在 **Today** 之后，但优先级为 **高** 或 **最高** 的任务

```tasks
due after 2026-03-31
not done
(priority is high) OR (priority is highest)
hide task count
```
